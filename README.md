# 🧬 Rosalind Bioinformatics Solutions

My personal repository containing Python solutions to the bioinformatics challenges from [Rosalind.info](https://rosalind.info/).

## 🚀 About Rosalind
Rosalind is a platform for learning bioinformatics and programming through problem-solving. The challenges cover algorithms, string manipulation, combinatorial analysis, and graph theory applied to biological data.

## 📂 Repository Structure
All solutions are written in **Python 3** and are located in the `solutions/` directory. The files are named after the specific Problem ID given on the Rosalind platform (e.g., `DNA.py`, `RNA.py`).
